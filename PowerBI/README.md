The Power BI in this folder is free for download and use as part of the showcase project for the Garage Tracker Pro mockup. The attached images provide a sample of what you'll see once you download the report.

Follow these steps to connect this report to your SQL Server data warehouse once the warehouse has been created:
1) Right-click any data source in the table catalog to the right and select "Edit Query" to access Power Query Editor
2) Click on "Manage Parameters"
3) Change the server parameter to match your local SQL Server name

The two attached images show the pages that are present in the Power BI file if you are just browsing this repository.
